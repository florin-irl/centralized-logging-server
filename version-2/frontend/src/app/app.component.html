Charlie Service

How to Use

1. Fill in Phone ID and Phone Number.
2. Choose how many entries to add in Number of Entries:
  • If set to 1 → Adds only the specified entry.
  • If greater than 1 → Adds the specified entry and the rest are randomly generated.
3. Click SEND to send data to the respective service.
4. Click CLEAR to reset the form fields.
